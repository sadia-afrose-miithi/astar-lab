# A* Search Algorithm Lab

## Description
This project implements the A* search algorithm to find the shortest path in a 2D grid.

## Features
- Uses Manhattan distance as heuristic
- Avoids walls (1)
- Finds optimal shortest path

## Input Format
- First line: rows and columns
- Next lines: grid (0 = empty, 1 = wall)
- Next line: start position
- Last line: target position

## Output
- Path found with cost
- Shortest path OR no path found
